Declaração de variáveis


    // variavel que funciona em todo package
    var w = "Variavel package level escope"
    
    func main() {
        /*variaveis que funcionam apenas nesse codeblock e
        apenas se forem declaradas antes de serem chamadas*/
        x := 10
        y := "Olá, Bom dia!"
        z := false
    
        //utilizado o comando Printf (print formatado)
        //%v = Valor
        //%T = Tipo
        fmt.Printf("x: , %v, %T\n", x, x)
        fmt.Printf("y: , %v, %T\n", y, y)
        fmt.Printf("z: , %v, %T\n", z, z)
    
        x, j := 20, 30
        fmt.Printf("x: , %v, %T\n", x, x)
        fmt.Printf("j: , %v, %T\n", j, j)
        fmt.Printf("w: , %v, %T\n", w, w)
    
    }   


Output:

            
        x: , 10, int
        y: , Olá, Bom dia!, string
        z: , false, bool
        x: , 20, int
        j: , 30, int
        w: , Variavel package level escope, string

Declaração, inicialização e atribuição

        var x int //isso é uma declaração
        
        func main() {
            x = 10 //isso é uma inicialização
            fmt.Printf("%v, %T \n", x, x)
            x = 20 //isso é uma atribuição
            fmt.Printf("%v, %T", x, x)
        }


Output:

        10, int 
        20, int

O Pacote fmt

Existem dois tipos de Strings na liguagem GO, as interpreted strings literals e as raw string literals, veja a seguir um exemplo de cada:


        func main() {
            x := "Isso é \n uma \t interpreted strings"
            fmt.Printf("%v, %T \n", x, x)
            y := `"Isso é  uma  raw strings"`
            fmt.Printf("%v, %T \n", y, y)
        }

As raw Strings são decalradas entre ` e não são interpretadas


Função Sprintln

A função Sprintln Concatena strings, mas não exibe nada na tela.

        func main() {
            x := "Oi "
            y := "Bom dia"
            z := fmt.Sprintln(x, y)
            fmt.Printf(z)
        }

Output:

        Oi  Bom dia
