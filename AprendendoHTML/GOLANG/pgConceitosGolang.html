Conceitos em GoLang

Eficiência

A linguagem GoLang visa a eficiência, portanto se você declarar uma variável e ela não for utilizada em nenhuma parte do programa, a linguagem interpretará como ineficiência e irá gerar um erro. Para que isso não ocorra, podemos usar o "_" para descartar o valor, assim não irá ser gerado o erro.

Declaração e atribuição de Variáveis

Em Go, utilizamos o operador ":=" (gopher) para declarar e atribuir um valor inicial a uma nova variável. O Go definirá automaticamente o tipo da variável com base no valor fornecido durante a declaração. No entanto, é importante notar que este operador só pode ser usado para declarar uma nova variável, não para atribuir um valor a uma variável já existente, para isso utilizamos o operador "=". Um detalhe importante é que o gopher funciona apenas em codeblocks, para declarar uma variável "global" deve ser utilizado o comando "var x = 1" fora de um codeblock
