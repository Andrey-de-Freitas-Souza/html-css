Conceitos em GoLang

Eficiência

A linguagem GoLang visa a eficiência, portanto se você declarar uma variável e ela não for utilizada em nenhuma parte do programa, a linguagem interpretará como ineficiência e irá gerar um erro. Para que isso não ocorra, podemos usar o "_" para descartar o valor, assim não irá ser gerado o erro.

Declaração e atribuição de Variáveis

Em Go, utilizamos o operador ":=" (gopher) para declarar e atribuir um valor inicial a uma nova variável. O Go definirá automaticamente o tipo da variável com base no valor fornecido durante a declaração. No entanto, é importante notar que este operador só pode ser usado para declarar uma nova variável, não para atribuir um valor a uma variável já existente, para isso utilizamos o operador "=". Um detalhe importante é que o gopher funciona apenas em codeblocks, para declarar uma variável "global" deve ser utilizado o comando "var x = 1" fora de um codeblock

Constantes

São valores imutáveis no programa, podem ser úteis por exemplo para guardar o endereço do banco de dados. As constantes não possuem um tipo definido até elas serem utilizadas, diferentemente das váriaveis que possuem um tipo definido na sua declaração

Tipos de dados

A linguagem Go é uma linguagem que possui Tipos Estáticos, ou seja quando uma variável é declarada, ela terá o mesmo tipo do início até o fim do programa. Quando se atribui um tipo a uma variável no package level escope e não é atribuído um valor a ela, o valor só poderá ser atribuído dentro de codeblock.

Tipos Primitivos


  • Int
  • Float
  • String
  • Boolean

Tipos Compostos

A linguagem go possui os seguinte Tipos Compostos:

  • slice
  • array
  • struct
  • map

Esses tipos são criados pelo usuário, o ato de definir, criar, estruturar tipos compostos chama-se composição.

Valores iniciais de cada tipo:


  • Int: 0
  • Float: 0.0
  • Bollean: False
  • Strings: ""
  • Pointer, Function, interfaces, slices, channels, maps: nil

Overflow


Quando é definida com um valor maior que limite dela, o código gera erro. Caso ela atinja o valor limite e seja adicionado mais valores a ela ,ela zera como um odômetro.
