HTML

Cabeçalho em HTML:

<!DOCTYPE html> : Define o tipo de documento como HTML5.

<html lang="pt-br"> : Inicia o elemento HTML e define o idioma.

<head> : Inicia o cabeçalho do documento HTML

<meta charset="UTF-8"> : Especifica que suporta a maioria dos caracteres e idiomas.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> : Define a configuração da viewport

<link rel="shortcut icon" href="html.ico" type="image/x-icon">: Define o favicon

<title>Favicon</title >: Define o título do documento

</head > : Encerra a seção do cabeçalho

Códigos básicos em HTML:

<h1> .. <h6> : Títulos ordem semântica

<p> : parágrafo

<br> : pula uma linha

<hr> : faz uma linha

Códigos Semânticos:

<code> : usado para códigos-fonte

<pre> : texto formatado

<blockquote cite = "Andrey"> : citações

<abbr title = "Nome completo"> : abreviações

<strong> : negrito

<em> : usado para dar ênfase

<mark> : marca o texto

<del>: risca o texto

<ins>: sublinha o texto

<sub>: texto subscrito

<sup>: texto sobrescrito

Iframes

Código base

<iframe src="https://www.youtube.com/" frameborder="0"></iframe>

obs: Os iframes podem ter nomes para dar referencia a links.

srcdoc

permite usar html para a "capa" do iframe.

srcdoc="<h1> Escolha uma das opções acima"</p>

sandbox

protege para que os sites não captem dados.

sandbox="allow-same-origin allow-forms allow-scripts" permite que apenas páginas so seu site permita cadastro de dados

referrerpolicyp

bloqueia o outro site de pegar informações

referrerpolicy="no-referrer"
