Cabeçalho em HTML:

<!DOCTYPE html> : Define o tipo de documento como HTML5.

<html lang="pt-br"> : Inicia o elemento HTML e define o idioma.

<head> : Inicia o cabeçalho do documento HTML

<meta charset="UTF-8"> : Especifica que suporta a maioria dos caracteres e idiomas.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> : Define a configuração da viewport

<link rel="shortcut icon" href="html.ico" type="image/x-icon">: Define o favicon

<title>Favicon</title >: Define o título do documento

</head > : Encerra a seção do cabeçalho

Códigos básicos em HTML:

<h1> .. <h6> : Títulos ordem semântica

<p> : parágrafo

<br> : pula uma linha

<hr> : faz uma linha

Mais alguns códigos:

<code> : usado para códigos-fonte

<pre> : texto formatado

<blockquote cite = "Andrey"> : citações

<abbr title = "Nome completo"> : abreviações

Links

links externos

links internos

<a href = "Página desejada" target = "_self" rel = "next">Título do link</a>

links de downloads
