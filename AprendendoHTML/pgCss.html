Seletores

Id

O id é utilizado quando o estilo será utilizado apenas em 1 vez.

<h1 id = "principal">Criando sites com HTML e CSS/h1>

#principal{

font-size: 3em;

text-align: center;

background-color: blue;

color: white;

}

Class

A classe é usada quando o estilos for utilizados várias vezes.

<h1 id = "destaque">Criando sites com HTML e CSS/h1>

.destaque{

font-size: 3em;

text-align: center;

background-color: blue;

color: white;

}

Pseudos classes

Esse estilo será aplicado quando o cursor passar em cima da div:

<div:hover{

background-color: yellow;

cursor: pointer;

}

Background

Códigos básicos backgrounds

background-image: url('imagens/wallpaper001.jpg');

background-color: skyblue;

background-image: linear-gradient(to bottom, yellow, red);

background-image: url('imagens/pattern003.png')

Configurações de background

background-size: 100px 100px; (auto,contain,cover)

background-repeat: repeat-y; (repeat, no-repeat)

background-position: left top; (coluna,linha)

background-attachment: fixed; (fixed,cover)

Shorthand

color > image > position > repeat > attachment

background: black url('imagens/wallpaper002.jpg') center center no-repeat fixed;

background-size: cover;

Centralização

O container sempre deve ser relative e o conteúdo deve estar dentro do container.

#container{

background-image: url('imagens/target001.png');

background-size: 100% 100%;

position: relative;

height: 96vh;

padding: 10px;

}

O conteúdo sempre deve ser absolute.

#conteudo{

background-image: url('imagens/target001.png');

background-size: 100% 100%;

position: absolute;

height: 200px;

width: 400px;

left: 50%;

top: 50%;

transform: translate(-50% ,-50%);

}
